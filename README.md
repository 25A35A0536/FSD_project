A simple and user-friendly organic grocery shopping website that allows users to explore and view different grocery products like fruits, dry fruits, and beverages in a clean and responsive interface.

🚀 Features
🥕 Browse organic grocery products (fruits, dry fruits, beverages, etc.)
🛒 Cart page interface for selected items
📸 Gallery to showcase products
📞 Contact page for user communication
🚚 Delivery information section
📱 Responsive design for mobile and desktop
🌿 Simple and clean UI focused on organic products
🖥️ Demo

Try the project locally by opening the main file in your browser.

🔧 Installation

Clone the repository:

git clone https://github.com/25A35A0536/FSD_project.git
cd FSD_project

Open the project:

Double-click on index.html
OR
Use Live Server (recommended for better experience)
📝 Usage
Open the homepage (index.html)
Browse different product categories
Navigate through pages like cart, gallery, and contact
Explore product details visually
